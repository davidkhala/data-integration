# Informatica PowerCenter

## Workflow
“设计处理逻辑”：定义什么时候、按什么顺序、用什么资源来跑这些处理
### Session
真正调用 Mapping 的执行单元



### Command Task
执行 Shell / SQL


### Decision
条件分支
### Link
Conditional Link

Link Types
- Success: run when 上一个 Task 成功
- Failure: run when 上一个 Task 失败
- Always / Completion: run
- Condition: customized condition expression 
 

### Event / Timer



## Mapping
“执行与调度”：定义数据怎么处理（数据从哪来、怎么转换、到哪里去）
- Source: Data source
- Transformation: Data Transform
- Target: Data Sink
